## Steps to install and run the program

```sh
git clone https://github.com/parth2118/Java-Mini-Project.git
```

```sh
cd Java-Mini-Project/IT\ Calculator/
```

```sh
javac FinalEmployeeFH.java
```

java FinalEmployeeFH
